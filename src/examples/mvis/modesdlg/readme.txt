
				 ****************************************
				 * MegaVision Sample Program:  MODESDLG *
				 ****************************************

Supported environments:
	16 bit DOS real mode
	32 bit DOS protected mode
	32 bit Windows
	32 bit Watcom Win386 extended Windows

Color depths supported:
	640x480 256 colors.

Description:
	This sample program shows how you can use the mode selector dialog
	box to allow the user to interactively select any of the available
	graphics modes on the fly.
