
                 ***************************************
                 * MegaVision Sample Program:  FILEDLG *
                 ***************************************

Supported environments:
	16 bit DOS real mode
	32 bit DOS protected mode
	32 bit Windows
	32 bit Watcom Win386 extended Windows

Color depths supported:
	640x480 256 colors.

Description:
    This sample program shows how you can use the file browser dialog boxes
    to allow the user to select load files, browse files and save files.
