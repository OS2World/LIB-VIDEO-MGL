
                 ***************************************
                 * MegaVision Sample Program:  GUIDEMO *
                 ***************************************

Supported environments:
	16 bit DOS real mode
	32 bit DOS protected mode
	32 bit Windows
	32 bit Watcom Win386 extended Windows

Color depths supported:
    All.

Description:
    This sample program demonstrates all the features of the MegaVision
    C++ GUI library, and basically combines the features of all the sample
    programs into a single demo.
