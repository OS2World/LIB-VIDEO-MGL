
                 **************************************
                 * MegaVision Sample Program:   HELLO *
                 **************************************

Supported environments:
    16 bit DOS real mode
    32 bit DOS protected mode
    32 bit Windows
    32 bit Watcom Win386 extended Windows

Color depths supported:
	640x480 256 colors.

Description:
	This sample program is the simplest MegaVision program that shows you
	how to initialise the library, and display a simple menu bar and
	message box.
